# Bert-text-classification
Amazon产品分类预测系统，对⽐多种机器学习和深度学习技术预测结果，使用BERT+CNN对比RNN、RCNN等多种方式提高模型性能 

### 结果分析

| 模型        | Test_Acc |
| ----------- | -------- |
| BERT        | 88.54%   |
| BERT+CNN    | 88.34%   |
| BERT+RNN    | 87.42%   |
| BERT+RCNN   | 88.01%   |
| BERT+DPCNN  | 88.40%   |
| ERNIR       | 94.09%   |
| ERNIR+DPCNN | 94.20%   |
